# Claude skill catalog

This folder contains **Claude-oriented skills** in the simple `skills/<skill-name>/SKILL.md` format.

## Skills

- `design-specs-writer/`
- `competitive-design-audit/`

## Install / use in Claude

In Claude, add skills by uploading each skill folder (or your packaged `.skill` files if you export them that way).

If you're using Cursor instead, see `.cursor/skills/` for the Cursor-optimized versions.

# Cursor Designer Agents — Skills

This folder contains **Cursor-first (Claude-compatible)** design skills you can paste into a Claude Project, or use as structured prompts in Cursor.

## Included skills

- `ux-audit/`: heuristic UX audits with prioritized fixes (P0–P2)
- `accessibility-review/`: WCAG-informed findings + remediation + regression checklist
- `component-spec-writer/`: engineering-ready component specs (variants/states/token hooks/a11y/QA)
- `design-system-governance/`: governance starter pack templates (workflow/DR/release notes)
- `research-synthesis/`: themes → insights → opportunities with evidence + hypotheses
- `analytics-insights/`: funnels/events → ranked UX hypotheses + instrumentation gaps
- `design-specs-writer/`: structured, engineering-ready design specs for components/flows
- `competitive-design-audit/`: decision-oriented competitive/comparative audits
